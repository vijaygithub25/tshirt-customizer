# T-Shirt Customizer

A simple and responsive web application that allows users to customize T-shirt text with a real-time preview.

---

## 🚀 Features

-  Add custom text to the T-shirt  
-  Live preview updates instantly  
-  Change text color  
-  Adjust font size  
-  Text alignment (left, center, right)  
-  Fully responsive (works on mobile & desktop)  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)

## 📌 How It Works

1. User enters text in the input field  
2. Text appears instantly on the T-shirt  
3. User can:
   - Change text color  
   - Adjust font size  
   - Modify alignment  
4. UI updates in real-time without page reload
